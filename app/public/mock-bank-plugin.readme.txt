Mock Bank TLSNotary plugin.

The WASM file (mock-bank-plugin.wasm) is built from the tlsn-plugin workspace in this repo.

Build and copy to this folder:
  From repo root: bun run plugin:build
  (Requires extism-js CLI; see tlsn-plugin/README.md.)

Or build only (from tlsn-plugin/): bun run build
  Then manually copy tlsn-plugin/dist/mock-bank-plugin.tlsn.wasm to app/public/mock-bank-plugin.wasm.

Alternatively set VITE_TLSN_PLUGIN_URL in app/.env to a URL that serves the built WASM.

Until the plugin is built, "Prove with TLSNotary" will fail at runPlugin (extension will prompt for a plugin; use the built WASM URL).
